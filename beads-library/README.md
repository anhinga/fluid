Beads audio synthesis library in Processing 2.2.1

The Mishka_mod_Frequency_Modulation_02 is a modification of an example supplied with Beads library to add 
an old-fashioned frequency modulation video synthesis working in sync with frequency modulation audio synthesis.

Reference: https://github.com/jsa-aerial/DMM/blob/master/design-notes/Late-2017/following-audio-synthesis.md

See the beginning of this document for Beads reference, and
the end section "The case of analog video synthesizers, oscilloscopes, etc."
for the ideology underlying this example. The example is created on September 28, 2017.
